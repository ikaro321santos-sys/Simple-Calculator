Simple Calculator (C#)

A simple console calculator built in C#.

Features

- Basic operations: addition, subtraction, multiplication, division
- Input validation using "TryParse"
- Loop system (can repeat calculations)
- Uses "enum" for operation handling

How it works

The program:

1. Asks for two numbers
2. Asks which operation to perform
3. Calculates and shows the result
4. Lets the user choose to continue or exit

Technologies

- C#
- .NET Console Application

What I learned

- Using "enum" to organize logic
- Handling user input safely
- Structuring a loop with "do-while"
- Separating logic into methods/classes

How to run

1. Open the project in Visual Studio
2. Run the program
3. Follow the instructions in the console
